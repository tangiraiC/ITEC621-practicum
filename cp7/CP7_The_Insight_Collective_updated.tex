\subsection{Executive Summary}\label{executive-summary}

This document prepares the Kogod Career Development LinkedIn profile
data for Question B: Skills Extraction. The workflow loads the cleaned
5k experience dataset, checks data quality, combines relevant profile
text fields, preprocesses the text, creates TF-IDF features, generates
weakly supervised label columns from the self-reported \texttt{Skills}
field, and exports train, validation, and test files for CP8 modeling.

\subsection{Part 8: Notes for CP8}\label{part-8-notes-for-cp8}

The train, validation, and test CSV files created above are the modeling
inputs for CP8. Each exported file contains:

\begin{itemize}
\tightlist
\item
  \texttt{linkedin\_url} as the profile identifier
\item
  TF-IDF feature columns
\item
  metadata columns prefixed with \texttt{meta\_}
\item
  label columns prefixed with \texttt{label\_}
\end{itemize}

In CP8, the team can select one label column at a time, such as
\texttt{label\_data\_analysis} or \texttt{label\_project\_management},
and compare multiple candidate models on the same held-out split.
